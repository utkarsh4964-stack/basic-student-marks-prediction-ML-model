Project Overview

This project implements a Supervised Machine Learning model to predict students' marks based on the number of hours they study using Simple Linear Regression.

It demonstrates strong fundamentals in:

Data preprocessing

Model training

Performance evaluation

Result interpretation

🚀 Key Highlights

📊 Performed data visualization to analyze correlation between study hours and marks

🔍 Implemented Train-Test Split for unbiased model evaluation

🤖 Built model using LinearRegression from Scikit-learn

📈 Evaluated performance using:

R² Score

Mean Squared Error (MSE)

📉 Plotted regression line to visualize prediction behavior

🛠 Technical Stack

Python

NumPy – Numerical computations

Pandas – Data handling & preprocessing

Matplotlib – Data visualization

Scikit-learn – Model building

Jupyter Notebook

🧠 Learning Outcomes

Through this project, I gained practical understanding of:

Supervised Learning

Real-world ML workflow
